Metadata data model


About the data model

This is a simplified data model for Metadata, based on the Metadata database used by the British Geological Survey to meet international spatial metadata standards such as the European INSPIRE Directive or ISO 19115.

We have simplified the model because we want to highlight the essence of the database without presenting an overly complex design. We have removed features and tables that relate to BGS-specific concepts that may not be relevant to others.

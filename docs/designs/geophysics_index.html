Geophysics

Externally Developed Designs

  • The geophysical database, GERDA, is a tool hosted by Geological Survey of Denmark and Greenland (GEUS). Through the GEUS.dk website you can access details of the Standards and Guidelines used as well as TableDescriptions and a Data Model Diagram

  • XODB is an Open Database for exploration,geology and mining. It contains structures to hold geophysical data, the project wiki produces documentation on the application and its backend database design
